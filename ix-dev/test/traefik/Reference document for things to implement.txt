Traefik supports environment variables

--entryPoints.web.address=":80" becomes TRAEFIK_ENTRYPOINTS_WEB_ADDRESS=":80"

Traefik dashboard port can not be changed

Traefik_network_connector allows for dynamic network modification for traefik. Giving it access to the necessary containers.



Labels for proxying other containers

traefik.enable=true
traefik.http.routers.whoami.rule=Host(`whoami.docker.localhost`)
traefik.http.routers.whoami.entrypoints=websecure
traefik.http.routers.whoami.tls=true
traefik.docker.network=mynetwork

 - Use HTTP or HTTPS

  - BYOC (Bring your own Certificate)

  - TLS Challange
  - DNS Challenge
